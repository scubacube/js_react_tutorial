git init -- инициализация гит репозитория. после этого появляется папка репозитория .git

git config --local user.name "Ivan" -- конфигурация
git config --local user.email email@mail.com

git status

git add -A -- добавляем все файлы (ALL) -- файлы начинают отслеживаться после этого
затем пишем git status 

если после git add -A мы создали какой-то файл, можем его добавить так -- git add nameoffile.css

git commit -a -m"first commit"  -- коммит, то есть контрольная точка, на которой сохраняются все файлы в этом определенном состоянии
если после коммита были изменены какие-то файлы, то нужно сделать git status, мы увидим эти изменения
снова пишем git add -A , таким образом переводим измененные файлы в новое состояние 
и коммитим всё git commit -a -m"second commit"

git log -- история всех коммитов
